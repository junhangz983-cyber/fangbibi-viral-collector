# 飞书表结构

## 表名

`对比账号表`

## 固定字段

| 字段 | 说明 |
| --- | --- |
| 标题 | 帖子标题或正文标题 |
| 链接 | 原帖 URL |
| 平台 | `抖音` / `小红书` |
| 账号 | 作者昵称 |
| 发布时间 | 原始发布时间，统一成文本 |
| 点赞 | 点赞数 |
| 收藏 | 收藏数 |
| 评论 | 评论数 |
| 分享 | 分享数 |
| 开头钩子 | 前 1 句或前 20-30 字 |
| 爆点判断 | 人工判断或规则化结论 |

## 入库规则

- 一条内容只占一行
- 链接去重
- 低于门槛的内容不进表
- 字段不加新列
