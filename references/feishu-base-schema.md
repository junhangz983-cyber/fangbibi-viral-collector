# 飞书多维表格字段

每次写入前先运行字段列表命令，以实时返回为准。当前已验证字段如下：

| 字段 | 类型 | 写入规则 |
| --- | --- | --- |
| 标题 | 文本 | 搜索卡片标题 |
| 链接 | URL 文本 | 抖音原帖链接，唯一去重键 |
| 平台 | 单选 | 固定写 `抖音` |
| 对标账号 | 文本 | 搜索卡片作者；账号导入时填写账号名 |
| 点赞 | 数字 | 整数，不带“万” |
| 爆点 | 文本 | 规则结论或待复核说明 |
| 入库日期 | 日期 | 本次写入日期，显示为 `YYYY-MM-DD` |
| 搜索关键词 | 文本 | 一个或多个词，用 `｜` 分隔 |

## 写入规则

- 不存在的旧字段不写入，例如发布时间、收藏、评论、分享。
- 新增记录用批量创建；同一轮内的重复链接合并关键词，已存在链接跳过。
- 私聊摘要中的链接从实际写入数据读取，不能手工拼接。
